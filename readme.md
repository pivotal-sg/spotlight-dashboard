# Spotlight Dashboard
* Front end for the most kickass dashboard for devshops *

## Setup instruction

 - Make sure you are using Node v5.4.0
 ```
   nvm install v5.4.0
   nvm use v5.4.0
 ```

 - Install all the requirements
 ```
    npm install
 ```

 - Run the server
 ```
   npm server
 ```
 - view the latest version of the application at localhost:8080
